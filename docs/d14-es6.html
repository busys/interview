2. var、let、const之间的区别

var 有变量提升, 可重复声明, 不支持块级作用域, 可修改 let 无变量提升, 不可重复声明, 支持块级作用域, 可修改 const 无变量提升, 不可重复声明, 支持块级作用域, 不可修改

4. ES6的模板字符串有哪些新特性？并实现一个类模板字符串的功能

  1. 字符串格式化
  2. 变量拼接
  3. 多行拼接
const a = '113';
const str = `这是$[a]的字符串`;
function templateFn(str) {
  return str.replace(/\$\[([^]]*)\]/g, (_, val) => {
    // 相当于从全局中去找匹配到的变量
    return eval(val)
  })
}
templateFn(str)

8. setTimeout、Promise、Async/Await 的区别

事件循环中分为宏任务队列和微任务队列 setTimeout会在倒计时结束将任务加入宏任务队列 Promise会返回一个异步任务, 该任务是微任务会放入当前宏任务的微任务中 async方法执行时遇到await,将其后的任务放入微任务队列,让出执行栈,让同步代码先执行
